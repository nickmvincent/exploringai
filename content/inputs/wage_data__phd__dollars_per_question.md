# Expert benchmark rate per question

**Value:** 200 dollars per question

## Description

A conservative benchmark for paying expert contributors to produce evaluation questions.

## Key Assumption

Humanity's Last Exam offered a $500,000 prize pool and produced 2,500 public questions. Dividing the prize pool by the retained public questions gives roughly $200 per question, excluding organizer and reviewer labor.
