# Proposed settlement value per work (Anthropic books case)

**Value:** 3,000 dollars per work

## Description

Proposed average payout per identified work in the Anthropic books settlement.

## Key Assumption

This is a proposed settlement benchmark from an October 17, 2025 filing, not a court-declared market value for every copyrighted book.
